## 📌 Overview  
The Universal Bridge Architecture unifies identity, tokenization, document permanence, and deterministic cross‑ledger settlement across heterogeneous blockchain systems.  
This repository accompanies the full research archive published on Zenodo and provides a developer‑friendly entry point into the ecosystem.

Core components include:

- **Universal Scribe** — platform‑agnostic identity capsules, Merkle scribe trees, Taproot anchoring  
- **Canonical Tokenization** — deterministic document permanence and reversible state encoding  
- **Semantic Tokenization** — meaning‑preserving identity resolution  
- **URIB Canonical Architecture** — seven primitives and full system design  
- **Cross‑Ledger Anchoring Systems** — XRPL, Stellar, Bitcoin Taproot, Solana  
- **ISO 20022 Banking Bridge** — deterministic settlement alignment  
- **ThreadZero Mathematical Framework** — truth model, invariants, fractal binding equations
